# Externa API Övning

This project demonstrates how to work with external APIs and integrate AI using OpenAI in an ASP.NET Core Web API.

## Project Overview

The API performs the following steps:

1. Fetches posts from the JSONPlaceholder API
2. Converts JSON responses into C# models
3. Sends the posts to OpenAI for analysis
4. Returns structured JSON results

## Technologies Used

- ASP.NET Core Web API
- C#
- HttpClient
- JSON serialization
- OpenAI API

## What I Learned

Through this project I learned:

- How to call external APIs using HttpClient
- How to convert JSON responses into C# models
- How to structure a Web API project using Models and Services
- How to integrate OpenAI into a backend application
- How to handle API responses and errors
